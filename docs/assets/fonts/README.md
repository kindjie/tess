# Vendored fonts

- `fraunces-latin.woff2` — Fraunces variable (wght 300-600, latin subset),
  fetched from Google Fonts (family API version v38). Chosen as the
  closest open match to the Sirenia Light logo wordmark; used for site
  headings via `docs/stylesheets/extra.css`.
- License: SIL Open Font License 1.1 (`OFL-fraunces.txt`).
  Upstream: <https://github.com/undercasetype/Fraunces>.

`https://fonts.googleapis.com/css2?family=Fraunces:opsz,wght@9..144,300..600`
(send a modern browser User-Agent to receive woff2 URLs), then update this
note.
