# Repository History

This repository retains its pre-public pull requests as development records.
Pull requests #1 through #26 contain useful discussion and review context, but
their commits and diffs reflect history that was superseded before the public
release process began. They are not release provenance or an authoritative
description of the current implementation.

Pull request #27 established the canonical post-rewrite baseline. The current
branches, signed release tags, maintained documentation, and source code are
the authoritative project history. When an earlier pull request conflicts with
one of those sources, use the current source.

The retained pull requests are intentionally available for historical context.
Their commit identifiers do not need to appear in the canonical branch or tag
graph.
