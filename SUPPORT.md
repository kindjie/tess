# Support

Use [GitHub Discussions](https://github.com/kindjie/tess/discussions) for
questions, integration help, and open-ended design conversations.

Use [GitHub Issues](https://github.com/kindjie/tess/issues) for reproducible
bugs and focused feature requests. Include the tess version or commit, compiler
and CMake versions, platform, a minimal reproducer, expected behavior, and
actual behavior.

Report vulnerabilities privately according to [SECURITY.md](SECURITY.md).
